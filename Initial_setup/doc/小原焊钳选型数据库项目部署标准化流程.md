## **部署流程**  

### **1. 切换至项目目录**  
```bash
cd D:\OG2\SERVO GUN\SERVO GUN250616\V1.3.3
```
> **预期结果**：控制台路径显示为 `D:\OG2\SERVO GUN\SERVO GUN250616\V1.3.3>`  

---

### **2. 创建虚拟环境**  

#### 2.1.Windows环境

```bash
# 删除旧虚拟环境（如有可通过以下命令删除也可手动删除venv文件夹）
rmdir /s /q venv

# 创建新虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate
```
> **预期结果**：控制台前缀显示为 `(venv) D:\OG2\SERVO GUN\SERVO GUN250616\V1.3.3>`  

---

#### 2.1.Linux环境

```bash
# 删除旧虚拟环境（如果已存在）
rm -rf venv   # 强制递归删除 venv 文件夹，请慎用！

# 创建新虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate
```

> **预期结果**：控制台前缀显示为 `(venv) user@hostname:~/your-project-path$`



Linux环境如果运行时提示错误：`No module named venv`，请先安装：

```bash
sudo apt update
sudo apt install python3-venv
```

### **3. 安装Python依赖**  

```bash
pip install django==5.2.3
pip install pillow
pip install chardet
```
> **预期结果**：  
> ```bash
> Successfully installed asgiref-3.8.1 django-5.2.3 sqlparse-0.5.3 tzdata-2025.2
> Successfully installed pillow-11.2.1
> Successfully installed chardet-5.2.0
> ```
> ⚠️ **提示**：若安装卡顿，请开启VPN。  
>
> #### 永久国内镜像源配置（推荐开发环境使用）：
>
> 编辑或创建 pip.conf 文件（Linux/macOS 在 `~/.pip/pip.conf`，Windows 在 `%HOMEPATH%\pip\pip.ini`）：
>
> ```bash
> [global]
> index-url = https://pypi.tuna.tsinghua.edu.cn/simple
> trusted-host = pypi.org
>             files.pythonhosted.org
>             pypi.tuna.tsinghua.edu.cn
> ```

---

### **4. 数据库初始化**  
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py init_data
```
> **预期结果**：  
> ```bash
> No changes detected
> No migrations to apply.
> 数据初始化完成！
> 分类总数: 6
> 产品总数: 17202
> ```

---

### **5. 启动服务器**  
```bash
python manage.py runserver 0.0.0.0:8000
```
> **预期结果**：  
> ```bash
> Starting development server at http://0.0.0.0:8000/
> ```
