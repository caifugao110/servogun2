# 小原焊枪选型数据库使用协议

## 第一条 协议目的

本协议旨在规范小原焊枪选型数据库（以下简称"数据库"）的使用，保护技术资料的安全性和保密性，确保用户合理合法地使用本数据库及其包含的所有功能，包括但不限于AI搜索功能、产品查询、技术资料下载等。

## 第二条 适用范围

本协议适用于所有使用数据库的用户，包括但不限于小原（南京）机电有限公司的员工、合作伙伴、客户及其他经授权的使用者。

## 第三条 用户权利与义务

### 用户权利：
1. 在授权范围内查询和下载焊枪技术资料
2. 使用数据库进行产品选型和技术参考
3. 访问和使用数据库提供的AI搜索功能，获取智能产品推荐和技术支持
4. 获得技术支持和服务
5. 对数据库功能提出建议和反馈

### 用户义务：
1. 严格保护用户名和密码，不得泄露给第三方
2. 仅将技术资料用于产品选型，不得用于其他商业用途
3. 不得复制、传播或向第三方提供技术资料
4. 发现系统问题或安全隐患时及时报告
5. 遵守数据库的使用规范，不得进行任何可能影响数据库正常运行的操作
6. 合理使用AI搜索功能，不得输入违法、违规或无关内容

## 第四条 技术资料保密

1. 数据库中的所有技术资料均为小原（南京）机电有限公司的技术机密
2. 用户不得将技术资料用于逆向工程、仿制或其他侵权行为
3. 技术资料仅供参考，不作为任何法律依据
4. 用户应采取必要措施保护从数据库获取的技术资料的机密性

## 第五条 免责声明

1. 小原（南京）机电有限公司不保证数据库中技术资料的完全准确性
2. 因技术资料变更导致的任何损失，公司不承担责任
3. 用户应根据实际情况进行技术验证
4. 对于AI搜索功能提供的建议和结果，公司不承担直接责任，用户应自行验证其准确性
5. 公司不保证数据库服务的绝对连续性和可用性

## 第六条 违约责任

用户违反本协议的，小原（南京）机电有限公司有权：
1. 立即停止用户的访问权限
2. 要求用户承担相应的法律责任
3. 保留追究进一步法律责任的权利
4. 对于恶意使用AI搜索功能或其他数据库功能的用户，公司有权采取法律措施

## 第七条 协议变更

小原（南京）机电有限公司保留随时修改本协议的权利，修改后的协议将在数据库中公布。用户继续使用数据库即视为同意修改后的协议。

## 第八条 争议解决

因本协议产生的争议，双方应友好协商解决；协商不成的，提交南京市仲裁委员会仲裁。

## 第九条 协议生效

本协议自用户点击同意之日起生效。
